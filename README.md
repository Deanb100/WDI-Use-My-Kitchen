# Use My Kitchen

- **[Live demo](http://use-my-kitchen.herokuapp.com/)**
- User stories
- Wireframes

Like AirBnB but for finding kitchen and dining spaces for your next dinner party.

## Technologies used
- Ruby
- Ruby on Rails + ActiveRecord
- PostgreSQL
- HTML + CSS
- JavaScript + jQuery + AJAX

## Installation instructions
```
bundle install
rails db:create
rails db:migrate
rails s
```
